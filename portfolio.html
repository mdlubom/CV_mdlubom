JavaScript is the programming language of HTML and the Web. I can use JavaScript to manipulate the web page elements and to make it dynamic. I can use it to change HTML content, HTML styles, HTML attributes.

SQL (Structured Query Language) is a standard language for storing, manipulating and retrieving data in databases. With SQL I am able to communicate and manipulate databases such as to insert,select,update,delete certain information from a database according to a certain functionality in the website.




PHP(recursive acronym for PHP: Hypertext Preprocessor) is a server scripting language, and a powerful tool for making dynamic and interactive Web pages.
